### Context
Rust 프로그래밍 언어의 주요 특징을 이해하고, 다른 언어와 비교하여 Rust가 가진 장점을 파악하고자 한다.

### Core
Rust는 다음과 같은 주요 특징을 가진다.

*   **메모리 안전성 (Memory Safety without Garbage Collection)**: 가비지 컬렉터 없이도 메모리 누수(memory leak)나 dangling pointer와 같은 문제를 방지한다.
*   **소유권 시스템 (Ownership System)**: 각 값(value)에 대한 소유자를 명확히 하여 메모리 관리를 자동화한다.
*   **borrowing and References**: 데이터의 소유권을 옮기지 않고 참조를 통해 안전하게 접근한다.
*   **패턴 매칭과 Enums (Pattern Matching and Enums)**: 강력한 패턴 매칭 기능을 통해 데이터 타입을 안전하게 처리한다.
*   **에러 핸들링 (Error Handling)**: Result 타입을 사용하여 에러를 명시적으로 처리한다.

### Insight
Rust는 성능과 안전성을 동시에 추구하는 시스템 프로그래밍 언어이다. 가비지 컬렉션 없이 메모리 안전성을 보장하는 소유권 시스템은 Rust의 가장 큰 특징 중 하나이며, 이를 통해 예측 가능하고 효율적인 코드 작성이 가능하다. 다만, 소유권 및 borrowing 개념은 학습 곡선이 높을 수 있지만, 안정적인 시스템 구축에 필수적인 요소이다.
